3.1-1 IsIndecomposable
‣ IsIndecomposable( arg )( property )

Returns: true if the cycle set is indecomposable

Let \(X\) be a cycle set. We say that \(X\) is indecomposable if the group \(\mathcal{G}(X)=\langle\varphi_x:x\in X\rangle\) acts transitively on \(X\).
